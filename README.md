# ddtank-rs
An easy-to-use ddtank login tool, which supports many platform.

## Requirements
 - Sciter 5.0.1.4
 - Rustc 2021
 - ui.rc

### build ui.rc
```
path-to-sciter-js-sdk/bin/windows/packfolder.exe "./src/ui" "./src/ui.rc" -binary
```

## Current Supported Platform
 - 7K7K
